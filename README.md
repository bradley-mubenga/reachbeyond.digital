# Reach Beyond

This project is a custom WordPress theme created using SCSS and Advanced Custom Fields to display data on the front end.

## Prerequisites

Before running this project locally, make sure that you have the following installed:

- MAMP (or any other local server software)
- WordPress

## Installation

To install and run the project locally, follow these steps:

1. Start MAMP and navigate to the `localhost` in your web browser.
2. Click on the phpMyAdmin link to access the database management tool.
3. Create a new database for the project.
4. Download and install WordPress in the `htdocs` folder of your MAMP installation.
5. Navigate to the `wp-content/themes` folder within the WordPress installation.
6. Move the entire project folder to the `themes` folder.
7. In your WordPress dashboard, navigate to Appearance > Themes and activate the project theme.

## Usage

This project uses Advanced Custom Fields to display data on the front end. To modify the data displayed, you can use the Advanced Custom Fields plugin in the WordPress dashboard.

The SCSS files for this project are located in the `scss` folder. To compile the SCSS files into CSS, you can use a preprocessor like Sass or a task runner like Gulp.
